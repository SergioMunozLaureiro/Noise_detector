# Noise_detector
An Arduino-based smart acoustic monitoring system designed for classroom noise control. Features real-time sound amplitude processing using the KY-038 sensor, a visual LED traffic light system, and remote telemetry via Bluetooth (HC-05). Includes a physical engineering solution for passive amplification to enhance microphone sensitivity.
